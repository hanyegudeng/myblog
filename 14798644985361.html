Swift初试牛刀

2016/11/23 posted in  iOS

let:定义常量，可以在声明的时候不赋值，但是后面也只能赋值一次，第二次再赋值会报错
var:定义变量
定义常量或者变量的时候可以不指定数据类型，如果给定初始值，会在第一次指定的时候编译器会自动指定类型
例如：

var tmp=10//tmp编译器会指定为Int类型
tmp=10.099//会报错，让你转换成Int类型Int(10.099)

值不会隐式转换，如果需要转换数据，必须要显示转换
例如：

let aa:Float=90.09
let bb:Int=9
let cc:Float=aa+bb//编译器报错
let cc:Float=aa+Float(bb)//正确

创建数组和字典

var tmp=[1,3,4,5,"abc","def"] as [any]//如果都是字符串可以as [String]
var tmp1=["a":"b","c":"d"]//访问tmp1["a"]

//循环
for tmp in dd{
    print(tmp)
}

两个问号提供默认值操作：？？

var ee:String?="str"
ee = nil
print(ee ?? "hello")//打印hello

switch最后必须加一个default，否则报错

var color="red"
color="black"
switch color {
case "red":
    print("红色")
case "black":
    print("黑色")
default:
    print("啥颜色都不是")
}
//打印黑色

函数参数名前加“_“标示不适用参数标签

func test1(_ str:String,bb:String){
    print(str+bb)
}

test1("Hello", bb: "haha")

使用元组来让一个函数返回多个值

func test1(_ str:String,bb:String)->(min:Int,max:Int){
    return (10,100)
}

let result = test1("Hello", bb: "haha")
print(result.min)//打印10

函数的嵌套

func f1(str:String)->(String){
    print(str)
    var tmpStr = str
    func f2(tmp:String){
        tmpStr=tmpStr+" 1 "+tmp
    }
    f2(tmp: "hello")
    f2(tmp: "word")
    return tmpStr
}

f1(str: "CCC")//CCC 1 hello 1 word

函数作为返回值

func f3(_ tmpInt:Int) -> ((Int) -> Int) {
    var inttmp = tmpInt+1//11
    func f4(number: Int) -> Int {
        inttmp = inttmp+1//13
        return 1 + number + inttmp//21
    }
    inttmp = inttmp+1//12
    return f4//返回的是f4的函数
}
var ff = f3(10)//f4函数
ff(7)//21

函数做为参数传入

func f6(ff:(_ tmp2:Int)->(Int),tmp:Int)->(Int){
    return ff(tmp)+tmp//6+5=11
}

func f5(_ tmp:Int)->(Int){
    return tmp+1//5+1=6
}


f6(ff: f5, tmp: 5)//11

类

class testClass:NSObject{
    let tmp=10
    override init() {
        print("Init")
    }
    func test(_ str:String){
        print(str)
    }
    
    deinit {
        print("deint")
    }
}

var tc:testClass=testClass()
var tc2:testClass=testClass()
tc.test("tc")
tc=tc2//这个时候会调用deinit释放资源
tc.test("tc2")
